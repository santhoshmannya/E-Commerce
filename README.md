# Chronova - Luxury Watch E-Commerce Website
Overview

Chronova is a modern luxury watch e-commerce website designed with a premium black-and-gold aesthetic. The website showcases a collection of high-end timepieces with an elegant user interface, smooth animations, and a responsive design that delivers a seamless shopping experience across all devices.

Built using HTML, CSS, and JavaScript, Chronova demonstrates front-end web development skills, including responsive layouts, interactive components, local storage functionality, and modern UI design principles.
